# Food Application
This is a clean architecture app example that consume the api REST [spoonacular](https://spoonacular.com/food-api) and is built with

 - Coroutines
 - MVVM
 - Extension Functions
 - Dagger Hilt
 - Retrofit
 - Navigation Components with Safe Args
 - Cache Strategy (Repository Pattern)
 - ViewBinding
